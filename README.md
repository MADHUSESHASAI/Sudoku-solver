## üß† How It Works

1. User enters digits into the 9√ó9 grid.
2. Input validation ensures the puzzle follows Sudoku rules.
3. A **backtracking algorithm** explores possible number placements.
4. The system finds a consistent solution or reports if unsolvable.
5. Final solution is displayed instantly.
